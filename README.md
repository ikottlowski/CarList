# CarList
Simple application to evaluate Angular and .NET skills

# Quick Start
To run **CarList.Api** navigate to `/CarList.Api/`. Then, from the CLI, run the command `dotnet run`.

To run **CarList.Client** navigate to `/CarList.Client/`. Then, from the CLI, run the command `ng s`.

Once the Client and Api are both running. Navigate using your browser of choice to http://localhost:4200.
